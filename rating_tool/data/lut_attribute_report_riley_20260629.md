# LUT 属性拆解与 Riley 第一轮评分结论

生成日期：2026-06-29

## 方法

这次不按滤镜名字做场景映射。每个 LUT 先按颜色变换本身拆成属性：

- 风格：自然轻微、通用风格、色相风格化、胶片哑光、黑白/单色、特殊反转/负片
- 饱和：低饱和、略降饱和、自然饱和、略增饱和、高饱和
- 对比：低对比、略低对比、自然对比、略高对比、高对比
- 色相：中性色相、暖橙/金色、冷蓝/青色、冷青/青绿、偏洋红、偏绿、暖黄/偏绿

属性来自 LUT 对标准色块、肤色、天空/水、绿植、暖色食物、灰阶等样本的实际输出，不来自文件名。

## Riley 第一轮整体偏好

样本限制：Riley 当前有 10 张图、56 条滤镜评分，所以这里只能作为第一轮方向。

整体倾向：

- 风格：自然轻微、通用风格明显更稳；色相风格化得分下降明显。
- 饱和：自然饱和最好，其次略增饱和；低饱和整体偏弱。
- 对比：略高对比略优于自然对比；过低或过高对比暂时都不稳。
- 色相：中性色相最稳；偏洋红还能接受；暖橙/金色、冷蓝/青色要看场景；偏绿、暖黄/偏绿暂时表现差。

## 不同场景的第一轮优选逻辑

绿植/山林/花：

- 优先：自然饱和或略降饱和、自然对比或略高对比、中性色相。
- 可试：少量温和色相风格化，但不要大幅偏绿或偏黄绿。

天空/海边/水：

- 优先：自然轻微或通用风格、自然饱和、略高对比、中性色相。
- 暂不优先：重冷蓝、重青绿、低饱和灰蓝。

城市/建筑/街道：

- 优先：自然轻微或通用风格、自然饱和、略高对比、中性色相。
- 可保留少量：偏电影感的暖橙/冷蓝，但需要用户评分验证。

宠物：

- 优先：通用风格、自然饱和、自然或略高对比、中性色相。
- 暂不优先：低饱和、强色相风格化、偏绿/黄绿。这个结论也可暂时类比到肤色安全，但还需要人物样本补齐。

食物、商品/文档：

- 当前 Riley 样本太少，不能稳定判断。
- 先作为必备覆盖场景保留，但不要提前让食物暖调、商品冷净这类规则过早主导推荐。

## 重复与收敛方向

当前库的主要问题不是数量绝对多，而是集中在几个相近区域：

- 复古褪色/低饱和/自然对比重复较多。
- 日系清新/低饱和/偏洋红重复较多。
- 青橙电影感、暖橙复古、土色复古有多组近似。
- 特殊效果如负片、黑白、强风格化不应该进入默认评测主池，只适合特殊玩法池。

优先合并的近似组：

- `709复古褪色低饱和柔对比` / `709复古褪色极低饱和柔对比` / `709复古褪色极低饱和标准对比`
- `709复古年代2` / `709复古抬黑褪色细腻`
- `709复古土色柔和对比` / `709复古土色稳重对比`
- `709复古抬黑褪色清透` / `709复古抬黑褪色自然`
- `709复古年代3` / `709通用轻创意2`
- `709复古胶片褪色` / `709纪实中性标准`
- `709日系奶油低饱和` / `709电影胶片青橙强化`

## 建议的第一轮滤镜库结构

建议先把主评测池压到 16-24 个代表滤镜：

- 6-8 个自然/通用/中性色相/自然饱和/略高对比，作为主干。
- 3-4 个肤色安全候选，需要覆盖不同肤色与室内外光线。
- 3-4 个天空、水面、绿植友好的清透自然候选。
- 2-3 个暖橙/金色情绪候选。
- 2 个冷蓝/青橙电影感候选。
- 1 个黑白或特殊风格候选，放在特殊池，不参与默认推荐排序。

下一轮最需要补的是人物与肤色样本，尤其是多肤色、室内混合光、逆光/阴影脸。否则很容易把宠物或风景上的“好看”误判成适合人像。
